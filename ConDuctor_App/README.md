# 🚍 BusConnect – Smart Bus Ticket Booking Platform for TGSRTC HYDERABAD

BusConnect is a modern web application that helps users easily search for buses, book tickets, and manage their travel digitally.
The platform provides a smooth user experience for passengers to explore routes, view schedules, and reserve seats online.

🌐 **Live Website:** https://busconnect-meeticket.netlify.app <br>
📦 **GitHub Repository:** https://github.com/PradameshBingi/BusConnect

---

# ✨ Features

* 🔍 Search buses by route
* 🎫 Book bus tickets online
* 🧾 View booking details
* 🔐 Firebase authentication integration
* ☁️ Cloud-based backend
* ⚡ Fast UI powered by Next.js
* 🌍 Fully deployed with custom domain

---

# 🚀 Installation

Clone the repository:

```
git clone https://github.com/PradameshBingi/BusConnect.git
```

Go to the project folder:

```
cd BusConnect
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Open in browser:

```
https://busconnect-meeticket.netlify.app
```

---

# 🌍 Deployment

The application is deployed using **Netlify** with a custom domain.

Deployment steps:

1. Push code to GitHub
2. Connect repository to Netlify
3. Configure environment variables
4. Deploy automatically

---

# 🔒 Security

Firebase API keys are exposed on the client side but protected using Firebase security rules.
